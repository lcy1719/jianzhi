【技术栈】
①：架构: B/S、MVC
②：系统环境：Windows
③：开发环境：IDEA、JDK1.8、MySQL、Maven
④：技术栈：Java、MySQL、SpringBoot、Mybatis、Vue、拦截器

主要功能：
1.管理员：首页、个人中心、用户管理、企业管理、兼职信息管理、职位申请管理、留言板管理、系统管理
2.用户：首页、个人中心、职位申请管理，前台首页：首页、兼职信息、留言反馈、个人中心、后台管理
3.企业：首页、个人中心、兼职信息管理、职位申请管理等功能。
